Accessibility

This site implements a number of accessibility aids. Each menu item has a access key associated with it. For example, 'i' is associated with the 'Introduction' page.

The site allows the user to select from 3 different font-sizes (small, medium and large). The chosen size is persisted and will automatically be used the next time the user visits the site. The font size can also be changed by the browser.
